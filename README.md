# 🎯 Guess the Number Game 

A simple Python-based Guess the Number game where the computer randomly selects a number and the player tries to guess it in the fewest attempts possible. This project is beginner-friendly and demonstrates the use of loops, conditional statements, random number generation, and user input handling in Python.

## 📌 Project Overview

The Guess the Number Game is a console-based Python application. The program generates a random number within a specified range, and the player must guess the correct number. After each guess, the program provides hints such as whether the guessed number is too high or too low until the correct number is found.

## ✨ Features

- Random number generation
- Hints for higher or lower guesses
- Tracks the number of attempts
  
## 🛠️ Technologies Used
- Python 3
- Random Module

## ▶️ How to Run the Project

1. Clone the Repository
   
2. Navigate to the Project Directory
 
3. Run the Program

## 🎮 How to Play

- Run the Python program.
- The computer will generate a random number.
- Enter your guess.
- The program will tell you whether your guess is:
- Too High 📈
- Too Low 📉
- Continue guessing until you find the correct number.
- The program displays the total number of attempts taken.

Through this project, I learned:
